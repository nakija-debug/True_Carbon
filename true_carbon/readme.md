# TRUE CARBON - Digital MRV Platform

This is a professional landing page for TRUE CARBON, a digital Monitoring, Reporting, and Verification (MRV) platform for carbon credit projects.

## Features

1. **Modern, Responsive Design**: Works on all device sizes
2. **Interactive Data Visualizations**: Charts showing pollution and carbon data
3. **Login/Signup Functionality**: With demo credentials pre-filled
4. **Smooth Animations**: Elements fade in as you scroll
5. **Professional Color Scheme**: Green/teal theme matching environmental focus

## File Structure

- `index.html` - Main HTML structure
- `style.css` - All styling and responsive design
- `script.js` - Interactive functionality, charts, forms
- `assets/` - Folder for images (currently using Font Awesome icons)

## How to Edit

### 1. Change Content
- Open `index.html` to modify text content
- Update the features, solutions, or any text sections

### 2. Change Colors
- Open `style.css`
- Modify the CSS variables in the `:root` section at the top

### 3. Update Charts
- Open `script.js`
- Modify the data in the Chart.js sections (around line 140-220)

### 4. Add Images
- Place images in the `assets/` folder
- Update image references in `index.html` and `style.css`

### 5. Change Contact Information
- Open `index.html`
- Update the contact details in the footer section

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- Chart.js for data visualizations
- Font Awesome for icons
